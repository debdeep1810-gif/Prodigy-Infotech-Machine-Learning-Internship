# Prodigy Infotech Machine Learning Internship

This repository showcases the projects completed during my 4-week Machine Learning Virtual Internship at Prodigy Infotech.

## Projects Included

- Task 1: House Price Prediction using Linear Regression
- Task 2: Customer Segmentation using K-Means Clustering
- Task 3: Cats vs Dogs Classification using SVM
- Task 4: Hand Gesture Recognition
- Task 5: Food Recognition and Calorie Estimation

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- OpenCV
- Matplotlib
- Machine Learning
- Computer Vision

These projects helped me gain practical experience in data preprocessing, model building, training, evaluation, and real-world machine learning applications.

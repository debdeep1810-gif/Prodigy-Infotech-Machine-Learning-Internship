Cats vs Dogs Classification using SVM
Project Overview

This project implements a Support Vector Machine (SVM) model to classify images of cats and dogs. Images are preprocessed using OpenCV and transformed into feature vectors for classification.

Technologies Used
Python
OpenCV
NumPy
Scikit-Learn
Matplotlib
Machine Learning Algorithm
Support Vector Machine (SVM)
Workflow
Image preprocessing
Image resizing
Feature extraction
Model training
Prediction
Performance evaluation

Results
The SVM model achieved approximately 85% classification accuracy on unseen test images.
